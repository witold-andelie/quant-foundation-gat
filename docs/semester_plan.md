# Semester Project Plan

## Project Thesis

Build a Second Foundation-inspired quantitative energy trading research platform that demonstrates the full Data Engineering Zoomcamp skill set while keeping a runnable US equities alpha demo for portfolio presentation.

## Primary Track

The primary track is now energy and power-market research:

1. Ingest hourly market and forecast data.
2. Compute expression-based alpha signals around residual load, imbalance premium, and renewable forecast shifts.
3. Run batch feature generation with Spark.
4. Run streaming producers and consumers with Redpanda for intraday signal events.
5. Land curated data in a warehouse and expose marts through dbt.
6. Present diagnostics and backtests in Streamlit.

## Secondary Track

The secondary track remains US equities:

1. Demonstrate expression-based WorldQuant-style alphas.
2. Show IS/OOS diagnostics, uniqueness, and value-added checks.
3. Keep a fully local, recruiter-friendly demo.

## Deliverables

1. Reproducible local environment with Docker and Make.
2. Terraform for a cloud warehouse path.
3. Batch pipeline, streaming pipeline, orchestration flow, dbt marts, and dashboard.
4. Coverage audit showing alignment to Zoomcamp knowledge points.
5. Final report on robustness, uniqueness, value-added, and consistency.
